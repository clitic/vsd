## Getting Started

Add this to your Cargo.toml file.

```toml
[dependencies]
vsd-mp4 = "0.1.0"
```

Or add from command line.

```bash
$ cargo add vsd-mp4
```

See [docs](https://docs.rs/vsd-mp4) and [examples](https://github.com/clitic/vsd/tree/main/vsd-mp4/examples) to 
know how to use it.
